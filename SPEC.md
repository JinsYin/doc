# 书写规范

## 符号

| 符号                | 描述                                     | 示例                         |
| ------------------- | ---------------------------------------- | ---------------------------- |
| `X`                 | 文件名、文件路径、函数名、变量名、命令等 | `copy.c`                     |
| 【X】               | 按钮                                     | 【系统偏好设置】 -> 【网络】 |
| 《X》               | 书名                                     | 《Linux 命令》               |
| 「X」               | 专业术语                                 | 「系统调用」                 |
| “X”                 | 普通术语、句子                           | “术语”                       |
| （X）               | 解释，如英文注释                         |                              |
| <X>                 | 链接                                     |                              |
| [[X]]               | 链接                                     |                              |
| **X**               | 加粗                                     |                              |
| ***X***             | 加粗 + 斜                                |
| <ins>X</ins>        | 下划线                                   |                              |
| <br> 或 <br/>       | 换行                                     |                              |
| X<sub>n</sub>       |                                          |                              |
| <kbd>Ctrl + S</kbd> | 快捷键                                   |

## Encoding

* utf-8
