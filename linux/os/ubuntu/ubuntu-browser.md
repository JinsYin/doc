# Ubuntu Browser

## 安装

* chrome

> https://www.google.com/chrome/

* chromium

```bash
$ apt-get install chromium-browser
```

* Firefox

```bash
$ add-apt-repository ppa:ubuntu-mozilla-security/ppa
$ apt-get update
$ apt-cache policy firefox
$ apt-get install firefox # latest
```


## Flash 支持

```bash
$ apt-get install adobe-flashplugin
```
