# C 语言

## 概述

* 时间：1969年 ～ 1973年
* 地点：贝尔实验室
* 人物：丹尼斯·里奇、肯·汤普逊
* 特点：高效、灵活、功能丰富、高可移植性

## 历史

## 标准

* `K&R C` - 早期由 Ritchie 和 Ken 定义的标准
* `ANSI C` / `ISO C` / `Standard C` - 由美国国家标准协会（ANSI）为 C 语言制定的国际标准语法

## 编译器

* GCC（`gcc`、`g++`）
* Xcode
* Clang
* MSVC - Microsoft Visual C++
* ARM RealView

## 翻译过程

1. 预处理（preprocessing）
2. 编译（compilation）
3. 汇编（assembly）
4. 链接（linking）
