# 函数定义（Function definition）

函数定义提供了函数的实际函数体。

## 语法

```c
返回值类型 函数名(0个或多个参数声明)
{
    声明部分
    语句序列
}
```

```c
Return_Type Function_Name(Parameter_List) {
    Function_Body
}
```

* Return_type（返回类型）
  * 如果存在返回值，返回类型为函数返回值的数据类型
  * 如果没有返回值，返回类型为 **void**
* Function Name（函数名）
* Parameter List
  * 函数可能没有参数
* Function Body - 语句集
