# 浮点常量（Floating Constant）

## 浮点常量的组成

* 整数部分
* 小数点
* 小数部分
* 指数部分（包含一个 `e` 或 `E`、一个可选的带符号整型类型的指数）
* 类型后缀（`f`、`F`、`l` 或 `L`）

上面的每一部分都是可选的。整数和小数部分均由数字序列组成。

## 浮点常量的类型

浮点常量的类型由后缀确定：

| 后缀       | 类型        | 示例    |
| ---------- | ----------- | ------- |
| `f` or `F` | float       | 111f    |
| `l` or `L` | long double | 314E-5L |
| （无）     | double      | 3.14    |
