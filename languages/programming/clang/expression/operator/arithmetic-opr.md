# 算术运算符（Arithmetic Operator）

| 运算符 | 运算（定义）                                  | 示例        |
| ------ | --------------------------------------------- | ----------- |
| `+`    | * 加法（addition）<br> * 一元加（unary plus） | * `a = +a;` |
| `-`    | 减法或一元减                                  |             |
| `*`    | 乘法                                          |             |
| `/`    | 除法                                          |             |
| `%`    | 取余                                          |             |

C 语言没有提供幂（`**` 或 `^`）、阶乘（`!`）、求绝对值（`|x|`）等运算符。

如果某个算术运算符的所有操作数均为整型，则执行整型运算。但是，如果某个算术预算符有一个浮点型运算数和整型运算数，则在 **开始运算之前** 整型操作数将会被转换成浮点型。
