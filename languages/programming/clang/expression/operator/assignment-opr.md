# 赋值运算符（Assignment Operators）

| Operator | Syntax   | Same as     | Meaning |
| -------- | -------- | ----------- | ------- |
| `=`      | `a = b`  | ~           |         |
| `+=`     | `a += b` | `a = a + b` |         |
| `-=`     | `a -= b` | `a = a - b` |         |
| `*=`     | `a *= b` | `a = a * b` |         |
| `/=`     | `a /= b` | `a = a / b` |         |
| `%=`     | `a %= b` | `a = a % b` |         |
