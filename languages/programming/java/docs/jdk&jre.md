# JDK & JRE

| 对比项                          | 描述                                                                         |
| ------------------------------- | ---------------------------------------------------------------------------- |
| JRE（Java Runtime Environment） | JVM（Java Virtual Machine）的一种实现，运行编译完成的 Java 应用程序和 Applet |
| JDK（Java Development Kit）     | 包含 JRE，以及开发、编译 Java 应用程序和 Applet 的其他软件                   |
