# vary

vi. 变化，不同，不等
vt

## 短句

| 短句/短语            | 含义               |
| -------------------- | ------------------ |
| vary from ... to ... | 从 ... 到 ... 不等 |

## 例句

* A C program can vary from 3 lines to millions of lines.
* The name and version of text editors can vary on different operating systems.
