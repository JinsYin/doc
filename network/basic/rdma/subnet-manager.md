# 子网管理器

## 确定必要性

大多数 InfiniBand 交换机都带有嵌入式子网管理器，但如果需要币交换机固件更新的子网管理器或更完整的控制，RHEL7 包含 **opensm** 子网管理器。所有 InfiniBand 都必须运行子网管理器才能使网络正常运行。
