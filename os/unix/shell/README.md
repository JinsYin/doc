# Unix Shell

Shell 意为 `壳`。

## 类型

Unix 系统中主要有三类 shell （还有各种子分类）：

* Bourne shell (`sh`)：默认命令提示符是 `$`
  * Korn shell (`ksh`)
  * Bourne Again shell (`bash`)
  * POSIX shell (`sh`)
* C shell (`csh`)：默认命令提示符是 `%`
  * TENEX/TOPS C shell (`tcsh`) - BSD
* Z shell (`zsh`)

Bourne shell 是第一个出现在 Unix 系统中的 shell，所以被称为标准的 shell 。
