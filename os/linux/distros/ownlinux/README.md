# OwnLinux

## 流程

1. 内核编译
2. 获取基本的文件系统，比如debian或者redhat的
3. 启动，调试，得到一个基于 cli 的基本系统
4. 安装 gnome，xfce，kde 或者 lxde 等图形环境
5. 各种软件定制，安装
6. 记得改 LOGO
7. 发行，不要忽视版权问题，否则容易被鄙视。
