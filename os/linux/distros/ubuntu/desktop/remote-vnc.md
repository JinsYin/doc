# VNC 远程连接

当 Ubuntu 使用 vnc 开启远程连接时，由于其不支持 Ubuntu 原生桌面系统（`unity`），所以需要改用 `gnome` 桌面系统。

## 安装 gnome

```bash
$ sudo apt-get install gnome-panel gnome-settings-daemon metacity nautilus gnome-terminal
```
