# Ext4

_Ext4_（第四代扩展文件系统，fourth extended filesystem）是为 Linux 内核设计的一个日志文件系统。

## 大小限制

| 块大小  | 最大文件大小 | 最大文件系统大小 |
| ------- | ------------ | ---------------- |
| **4KB** | 16TB         | 1EB              |

## 特点

* 大型文件系统
* Extents
* 向后兼容
