# 文件时间戳

| 时间戳类型           | 查看方式 | 描述                             |
| -------------------- | -------- | -------------------------------- |
| atime（access time） | `ls -lu` | 文件的最后访问时间               |
| mtime（modify time） | `ls -l`  | 文件的最后修改时间               |
| ctime                | `ls lc`  | 文件状态（权限等）的最后改动时间 |

注：Linux 不会记录文件的创建时间
