# 文件时间戳

| 时间                       | 描述                                   |
| -------------------------- | -------------------------------------- |
| access time（actime）      | 访问时间；访问文件时更新               |
| status time（stime）       | 状态时间；文件状态（权限等）改变时更新 |
| modification time（mtime） | 修改时间；修改文件内容时更新           |
