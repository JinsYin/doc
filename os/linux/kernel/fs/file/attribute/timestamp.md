# 文件时间戳

| 时间戳类型 | 查看命令 | 描述                                   |
| ---------- | -------- | -------------------------------------- |
| atime      | `ls -lu` | 最近一次对文件的访问时间               |
| mtime      | `ls -l`  | 最近一次对文件的修改时间               |
| ctime      | `ls lc`  | 最近一次对文件状态（权限等）的改动时间 |

注：Linux 不会记录文件的创建时间
