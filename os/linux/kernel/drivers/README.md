# Linux 驱动（Linux Driver）

Linux 驱动（Driver）也叫 Linux 模块（module）。

每种设备类型都有与之相对应的设备驱动，用来处理设备的所有 I/O 请求。设备驱动属于内核代码单元，可以执行一系列操作，通常与相关硬件的输入或输出相对应。

* Display Driver
* Camera Driver
* Flash Memory Driver
* Binder（IPC） Driver
* Keypad Driver
* WiFi Driver
* Audio Driver

---

* Block Device Drivers
* Network Device Drivers
* Character Device Drivers
