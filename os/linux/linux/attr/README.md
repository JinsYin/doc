# 文件属性（file attribute）

系统调用 `stat()`、`lstat()` 和 `fstat()` 均可获取文件相关信息，但有所不同：

| 系统调用            | 描述                                                                                                |
| ------------------- | --------------------------------------------------------------------------------------------------- |
| [stat()](stat.md)   | 会返回所命名文件的相关信息                                                                          |
| [lstat()](lstat.md) | 类似 `stat()`，区别在于如果文件属于符号链接，`lstat()` 返回的是符号链接自身的信息（而非指向的文件） |
| [fstat()](fstat.md) | 返回由某个 _打开文件描述符_ 所指代的文件的相关信息                                                  |
