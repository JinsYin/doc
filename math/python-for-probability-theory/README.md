## 基础概率知识

* P(A) - A 发生的概率
* P(B) - B 发生的概率
* P(A | B) - B 发生的条件下，A 发生的概率
* P(B | A) - A 发生的条件下，B 发生的概率
* P(A \bigcap B) - B 同时发生的概率

如果 A 与 B 独立，则 `P(A) = P(A | B)`
