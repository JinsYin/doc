# Pool（存储池）

Pool：存储数据对象的逻辑分区。

## 组成

* Pool Type
* Placement Group
* CRUSH Ruleset -  Administrators set the CRUSH ruleset when creating a pool.
* Durability

> An administrator CANNOT change a pool’s ruleset after creating the pool.
