# hugo new 命令

## 用例

| 用例                                         | 描述     |
| -------------------------------------------- | -------- |
| `hugo new site <SITENAME>`                   | 新建站点 |
| `hugo new theme <THEMENAME>`                 | 新建主题 |
| `hugo new <SECTIONNAME>/<FILENAME>.<FORMAT>` |          |
| `hugo new post/<FILENAME>`                   |          |
