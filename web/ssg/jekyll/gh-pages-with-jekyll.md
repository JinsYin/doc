# 利用 Jekyll 建立 Github Pages 站点

Github Pages 内置支持 Jekyll，使得用户可以使用 Markdown 和 HTML 文件并选择合适的布局来建立完整的静态网站。

Github Pages 安装的是来自 `_config.yml` 的 `plugins` 指定的插件，而不是 Gemfile 指定的插件。
