# Google Chrome

## 安装

* Chrome

```bash
$ sudo apt-cache policy google-chrome-stable
$ sudo apt-get install google-chrome-stable=${YOUR-VERSION}
```

* Chromium

```bash
$ sudo apt-cache policy chromium-browser
$ sudo apt-get install chromium-browser=${YOUR-VERSION}
```
