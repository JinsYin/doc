# f.lux

## 安装

```bash
# Redshift 可以提供夜灯功能
$ sudo apt-get install redshift-gtk
```

```bash
sudo add-apt-repository ppa:nathan-renniewaldock/flux
sudo apt-get update
sudo apt-get install fluxgui
```

Dash 搜索 `flux` 即可
