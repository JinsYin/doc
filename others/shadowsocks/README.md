# Shadowsocks 

Shadowsocks 也通常简称 ss，支持 `sock4` 、`sock5`、`http`、`https` 等代理方式。


## 配置文件

* [gfwlist](./gfwlist.txt)：大陆白名单（需要使用 ss 的墙外网站列表）, [来源](https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt)
* [gui-config](./gui-config.json): ss 客户端配置
