## 命令

| 命令                  | 描述                          |
| --------------------- | ----------------------------- |
| `:w`                  | 保存                          |
| `:q`                  | 退出                          |
| `:q!`                 | 强制退出且不保存修改          |
| `:wq`                 | 保存修改并退出                |
| `:%s/原内容/现内容/g` | 全文替换 “原内容” 为 “新内容” |
