# Scala 从入门到放弃

Scala 是一门多范式的编程语言，有马丁奥的司机（Martin Odersky）在 2003 年发明。Scala 集成了面向对象和函数式编程语言的特性。


## What's scala?

Scala 是一门面向对象和函数式的多范式编程语言。

* Scala is object-oriented

* Scala is functional

> http://docs.scala-lang.org/zh-cn/overviews/
