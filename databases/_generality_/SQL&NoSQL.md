# SQL & NoSQL

## 概念

| 名称  | 描述                                      | 代表                                  |
| ----- | ----------------------------------------- | ------------------------------------- |
| SQL   | Structured Query Language，指关系型数据库 | Oracle、MySQL、PostgreSQL、SQL Server |
| NoSQL | Not Only SQL，泛指非关系型数据库          | MongoDB、Redis、Cassandra             |

RDBMS - Relational database management system
