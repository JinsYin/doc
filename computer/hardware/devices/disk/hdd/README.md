## 时间

```plain
总耗时 = 寻道时间 + 旋转延迟 + 传输时间
```

| 时间     | 描述                                 |
| -------- | ------------------------------------ |
| 寻道时间 | 磁头移动到相应磁道耗费的时间         |
| 旋转延迟 | 等待相应扇区移动到磁头下所耗费的时间 |
| 传输时间 | 传输请求块的时间                     |
