# 二进制（Binary）

## 形式

```c
+---+    +---+    +---+    +---+    +---+    +---+
|   |    |   |    |   |    |   |    |   |    |   |
+---+    +---+    +---+    +---+    +---+    +---+
  |        |        |        |        |        |
  |        |        |        |        |        +------> 1  的个数
  |        |        |        |        +---------------> 2  的个数
  |        |        |        +------------------------> 4  的个数
  |        |        +---------------------------------> 8  的个数
  |        +------------------------------------------> 16 的个数
  +---------------------------------------------------> 32 的个数
```

## 算术运算

| +     | 0   | 1   |
| ----- | --- | --- |
| **0** | 0   | 1   |
| **1** | 1   | 10  |

| ×     | 0   | 1   |
| ----- | --- | --- |
| **0** | 0   | 0   |
| **1** | 0   | 1   |
