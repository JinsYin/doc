# 八进制（Octal）

## 形式

```c
+---+    +---+    +---+    +---+    +---+    +---+
|   |    |   |    |   |    |   |    |   |    |   |
+---+    +---+    +---+    +---+    +---+    +---+
  |        |        |        |        |        |
  |        |        |        |        |        +------> 1     的个数
  |        |        |        |        +---------------> 8     的个数
  |        |        |        +------------------------> 64    的个数
  |        |        +---------------------------------> 256   的个数
  |        +------------------------------------------> 4096  的个数
  +---------------------------------------------------> 32768 的个数
```
