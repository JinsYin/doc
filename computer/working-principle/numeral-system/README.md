# 数字系统

数字系统（Numeral System），也叫记数系统（System of Numeration），是使用一组数字符号来表示数的体系。

* 中文数字（汉字数字）
* 罗马数字
* 阿拉伯数字

## 罗马数字

| 符号 | 数字 |
| ---- | ---- |
| I    | 1    |
| V    | 5    |
| X    | 10   |
| L    | 50   |
| C    | 100  |
| D    | 500  |
| M    | 1000 |

特点：

* 易于加减（以前的人们是这样认为的）：五个 `I` 是一个 `V`，两个 `V` 是一个 `X` ...
* 不利乘除
