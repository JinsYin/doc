# 十六进制（Hexadecimal）

## 形式

```c
+---+    +---+    +---+    +---+    +---+    +---+
|   |    |   |    |   |    |   |    |   |    |   |
+---+    +---+    +---+    +---+    +---+    +---+
  |        |        |        |        |        |
  |        |        |        |        |        +------> 1       的个数
  |        |        |        |        +---------------> 16      的个数
  |        |        |        +------------------------> 256     的个数
  |        |        +---------------------------------> 4096    的个数
  |        +------------------------------------------> 65536   的个数
  +---------------------------------------------------> 1048576 的个数
```
