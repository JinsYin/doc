# 四进制（Quanternary）

## 形式

```c
+---+    +---+    +---+    +---+    +---+    +---+
|   |    |   |    |   |    |   |    |   |    |   |
+---+    +---+    +---+    +---+    +---+    +---+
  |        |        |        |        |        |
  |        |        |        |        |        +------> 1    的个数
  |        |        |        |        +---------------> 4    的个数
  |        |        |        +------------------------> 16   的个数
  |        |        +---------------------------------> 64   的个数
  |        +------------------------------------------> 256  的个数
  +---------------------------------------------------> 1024 的个数
```
