# 十进制（Decimal）

## 形式

```c
+---+    +---+    +---+    +---+    +---+    +---+
|   |    |   |    |   |    |   |    |   |    |   |
+---+    +---+    +---+    +---+    +---+    +---+
  |        |        |        |        |        |
  |        |        |        |        |        +------> 1       的个数
  |        |        |        |        +---------------> 10      的个数
  |        |        |        +------------------------> 100     的个数
  |        |        +---------------------------------> 1000    的个数
  |        +------------------------------------------> 10000   的个数
  +---------------------------------------------------> 100000  的个数
```

## 示例

0以 4885 为例：

```plain
4885 = 4000 +
        800 +
         80 +
          5
```

```plain
4885 = 4 × 1000 +
       8 × 100 +
       8 × 10 +
       5 × 1
```

```plain
4885 = 4 × 10^{3} +
       8 × 10^{2} +
       8 × 10^{1} +
       5 × 10^{0}
```
