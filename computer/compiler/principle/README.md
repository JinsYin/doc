# 编译器原理

* 词法分析
* 语法分析
* 语义分析

## 编译示例

* hello.c

```c
#include <stdio.h>

int main()
{
    printf("Hello, world\n");
    return 0;
}
```

* 编译

```sh
$ gcc hello.c -o hello
```

## 编译流程

```plain
  hello.c   +--------------+     +-----------+  hello.s   +---------+  hello.o   +---------+
--------->  | 预处理器(cpp) | --> | 编译器(ccl) | ---------> | 汇编器(as) | ---------> | 链接器(ld) |
   源代码    +--------------+     +------------+            +---------+            +---------+

```
