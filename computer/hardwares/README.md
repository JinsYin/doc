## 内存（memory）

内存有三种类型：

* 缓存（Cache Memory） - 充当 CPU 和主存之间的缓冲（buffer）
* 主存（Main Memory）
  * RAM
  * ROM
* 辅存（Secondary Memory）/外存（External Memory）
  * Hard disk, CD, DVD
