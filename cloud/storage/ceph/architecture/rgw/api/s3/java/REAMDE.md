# S3 for Java

## 依赖

Maven：

```xml
<dependencies>
  <dependency>
    <groupId>com.amazonaws</groupId>
    <artifactId>aws-java-sdk-s3</artifactId>
    <version>1.11.376</version>
  </dependency>
</dependencies>
```

SBT：

```sbt
libraryDependencies += "com.amazonaws" % "aws-java-sdk-s3" % "1.11.376"
```
