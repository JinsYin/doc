# Ceph

目前使用的 Ceph 发行版本为 `jewel`（10.2.x），该版本也是 CephFS 的第一个稳定版本，所以块存储（RBD）、对象存储（RadosGW）、文件系统（CephFS）都可以用于生产环境。
